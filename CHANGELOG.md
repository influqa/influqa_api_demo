# Changelog

All notable changes to the Influqa API Demo project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-23

### Added
- Initial release of Influqa API Demo
- Comprehensive README with API documentation
- JavaScript examples for:
  - Authentication (login, register, refresh, logout)
  - User management (profile, updates, admin functions)
  - Offers (create, update, search, AI search)
  - Orders (create, manage lifecycle, messaging)
  - Coins (balance, purchase, transactions, gifts)
- Python examples for:
  - Authentication with reusable auth class
  - Offers management
  - Orders management
- cURL examples for:
  - Authentication
  - Offers operations
  - Orders management
  - Analytics
- Documentation files:
  - LICENSE (MIT)
  - CONTRIBUTING.md
  - CHANGELOG.md
  - .gitignore

### Features
- JWT Bearer Token authentication
- Rate limiting support
- Multi-language examples (JavaScript, Python, cURL)
- Comprehensive error handling examples
- Admin endpoint examples
- AI-powered search examples
- Webhook documentation

## Version History

### Versioning Scheme

We use [Semantic Versioning](https://semver.org/):
- **MAJOR** version for incompatible API changes
- **MINOR** version for added functionality (backwards compatible)
- **PATCH** version for backwards compatible bug fixes

### Support Policy

- Current major version: Full support
- Previous major version: Security fixes only
- Older versions: No support
